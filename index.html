This page documents the core visual system for my portfolio website – including colours, typography, buttons and key components.

Buttons

Buttons use the primary blue for clear calls to action, with hover states using the secondary shade. This keeps actions consistent and visually strong.

Primary Button Outline Button

Primary buttons are used for key actions like “View Projects” or “Contact Me”. Outline buttons are used for secondary actions such as “Download CV” or “View GitHub”.

Components – Card Example

Cards are used to present projects or experience in a structured, scannable way. Background panels and subtle shadows help them stand out from the page.

Usage Guidelines

• Use cards for projects, experience blocks, or key skills.
• Keep copy short and focused – title, 1–2 lines of meta, and a short description.
• Always align card buttons with the global button styles.
• Maintain consistent spacing between cards (e.g. 24–32px).
